# AZERTY Keymap for Mac

French AZERTY keyboard layout files for macOS.

## Files

| File | Description |
|------|-------------|
| `Français.dmg` | Standard French AZERTY layout |
| `Français - numérique.dmg` | French AZERTY layout with numeric keypad support |

## Installation

1. Download the `.dmg` file that matches your needs
2. Open the `.dmg` and run the installer
3. Go to **System Settings > Keyboard > Input Sources**
4. Add the new French AZERTY layout
5. Switch layouts using the menu bar icon or keyboard shortcut (default: `Control + Space`)

## When to Use

- **Français.dmg** — Standard AZERTY for MacBooks and Apple keyboards
- **Français - numérique.dmg** — AZERTY with extended numeric keypad support (for external keyboards)

## Requirements

- macOS 10.12+ (Sierra or later)
